🔗 Conexión Power BI

Conecta este sistema con Microsoft Power BI usando el endpoint de datos:

GET /api/v1/ecoseam/actividades?format=json&desde=2024-01-01
En Power BI Desktop: Obtener datos → Web → JSON y pega la URL del endpoint.
