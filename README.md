# ECOSEAM Operations — Sistema de Gestión Operativa

**Ecología y Servicios Ambientales S.A.C. (ECOSEAM)**  
PWA para registro de actividades, restricciones operativas y dashboard gerencial.

## 🚀 Acceso rápido
Abrir `index.html` en el navegador o instalar como app en el celular (PWA).

**Login:** Ingresa tu nombre de usuario asignado por el administrador.  
Usuarios de prueba predefinidos: `juan.perez`, `maria.garcia`, `carlos.lopez`, `r.torres`, `a.castro`, `admin`

## 📋 Roles
| Rol | Acceso |
|---|---|
| **Operador** | Registro de actividades propias |
| **Supervisor** | Dashboard + actividades + restricciones + revisión |
| **Gerencia** | Dashboard + reportes (solo lectura) |
| **Administrador** | Todo + gestión de usuarios |

## 🌿 Tecnología
- PWA (Progressive Web App) — funciona sin internet
- HTML/CSS/JS puro — sin dependencias externas
- Instalable en Android e iOS como app nativa

## 📦 Despliegue en GitHub Pages
1. Subir estos archivos al repositorio
2. Ir a **Settings → Pages → Branch: main → Save**
3. URL pública: `https://TU-ORG.github.io/ecoseam-ops/`
